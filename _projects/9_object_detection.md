A lightweight real-time object detection system using **MobileNet-SSD** and OpenCV, deployed on edge
hardware (Raspberry Pi, NVIDIA Jetson Nano). An earlier driver-assistance variant reached **72.7%**
mean average precision with real-time alerting.
